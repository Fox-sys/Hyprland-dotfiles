# Мой билдер для Hyprland

## Установка

- Установить Arch
- Включить multilib
- Отредактировать файл `/etc/locale.gen` и раскоментировать `ru_RU.UTF-8`
- `sudo locale-gen`
- `sudo pacman -S git python3 poetry`
- `git clone https://github.com/Fox-sys/Hyprland-dotfiles.git`
- `cd Hyprland-dotfiles`
- `./install.sh`
- `Hyprland`
